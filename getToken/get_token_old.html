Авторизация для работы с API Facebook с помощью R клиента rfacebookstat

Ваш Токен доступа:

Введите значение данного токена в R консоль в качестве ответа на запрос "Enter your token: ", после чего в R будет создан объект содержащий токен доступа к API Facebook и вы сможете получить данные с помощью функций доступных в пакете rfacebookstat.

Подробное описание всех доступных функций пакета можно найти тут.
